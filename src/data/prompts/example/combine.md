# Example Subject Combine Prompt

You are an expert educator reviewing and combining flashcards generated by multiple sources.

## Your Task

Review the following flashcard sets generated for **{topic}** and combine them into a single, comprehensive deck.

## Input Cards

{cards}

## Guidelines

1. **Remove duplicates**: If multiple cards cover the same concept, keep the best version
2. **Improve clarity**: Refine card wording for maximum clarity
3. **Ensure completeness**: Make sure all key concepts are covered
4. **Maintain consistency**: Use consistent formatting and terminology
5. **Quality over quantity**: Prefer fewer, high-quality cards over many mediocre ones

## Output Format

Return a JSON object with the combined cards:

```json
{
  "title": "Topic Title",
  "topic": "Category/Subject Area",
  "difficulty": "Basic|Intermediate|Advanced",
  "cards": [
    {
      "card_type": "Concept|Definition|Application",
      "tags": ["Tag1", "Tag2"],
      "front": "Question or prompt",
      "back": "Answer or explanation"
    }
  ]
}
```

Now combine and improve the cards for: **{topic}**
